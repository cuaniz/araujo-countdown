# Araujo Contract Countdown

Live countdown to Ronald Araujo's FC Barcelona contract expiry on June 30, 2031.

## Features

- Countdown timer (years, months, days, hours, minutes, seconds)
- Real-time cost calculation (EUR and USD)
- Player contract information
- Responsive design for mobile and desktop

## Built With

- HTML
- CSS
- JavaScript

## How to Use

1. Clone the repository
2. Open `index.html` in your browser
